# JLCEDA Personal Component Inventory

A modular personal component inventory extension for JLCEDA / EasyEDA Professional V3. Version `0.1.0` can look up LCSC parts, record exact or estimated stock, mark parts depleted, import CSV/JSON order files, rank in-stock parts first, copy catalog parts to common libraries, and place inventory parts in a schematic.

The extension does not require a custom server. It stores data through the official extension user-configuration API. Official documentation does not guarantee cross-computer synchronization, storage limits, propagation, or conflict behavior, so two-device validation is still required before presenting this as guaranteed cloud sync.

See [README.md](README.md) for the complete feature, build, installation, and limitation notes.
